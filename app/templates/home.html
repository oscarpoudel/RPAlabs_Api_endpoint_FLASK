Api Endpoints in App

/upload uses post method to upload video files. Validate video size, extension and length before upload and display response in form of json
/files uses get method to show all the files upload in server. Show response of file list in json format. We can use parameter name or max_duration or max_size for filter ps we can still implement upload date and file type as filter parameters but in this case I did'nt implement it
/charge uses post method with from elements: length size and type and validate parameters to provide response as charge in json
